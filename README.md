## 📌 What This Project Does
- Loads and preprocesses historical stock market data
- Builds and trains an RNN (LSTM) model for trend forecasting
- Visualizes predicted vs actual stock prices

---

## ▶️ How to Run
pip install -r requirements.txt
jupyter notebook stock_price_forecasting_rnn.ipynb
